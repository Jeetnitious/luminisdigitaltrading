Crypto Planet - Crypto Trading Exchange UI Template In Figma

By: Mad Brains Technologies LLP

Crypto Planet is a Crypto trading exchange UI template to sell, purchase, and hold crypto currencies. This template is designed with a lot of research and after creating a well researched UX. This enabled us to create the full Crypto trading exchange process. Crypto Planet offers a very unique, interactive and modern UI along with a well researched UX. The template has been designed in Figma which makes it easy to customize.

FEATURES

  • Template comes with both dark and light interfaces
  • It comes with 33 different pages that covers the whole trading exchange process
  • Template also consists 33 mobile UI interfaces

To Change a Logo

Please follow the below mentioned steps to change the logo.

Step 1: Go to Component folder that is present in the left bar of the Figma file.

Step 2: Go to the file "Web Components". You'll find the logo in the first row.. Once you'll update the new logo here, it will be updated in the whole template automatically.

To Make Changes In the Navbar & Footer

Navbar: In the second row of "Web Components" file, you'll find the different navbar designs. Whatever changes you'll make to these navbars will be automatically implemented to the rest of the template.

Footer: Go to Component folder that is present in the left bar of the Figma file.

Look for the file "Components Navbar & Footer" and make the desired changes for the universal impact in the template footers.

To Make Changes to the icons of the template

In the fiifth row of the "Web Components" file, you'll find all the icons used in this template.

To Make Changes to the mobile responsive elements of template

Please go to the same "Components" folder. Then look for the file "Mobile Components". In this file, you'll be able to find all the elements of the mobile responsive design of the template.
